# GEMINI.md — Antigravity / Google host pack

## Regole

- Leggi rules/workflows e skills del workspace prima di agire.
- Verifica il supporto MCP/auth del client reale prima di progettare un server remoto.
- Non assumere supporto generico a OAuth MCP senza test.
- Usa MCP Store o raw config secondo disponibilità.
- Per tool locali preferisci CLI/MCP locale; per governance cloud preferisci endpoint Google Cloud o server remoto governato.

## Percorso minimo

1. Compila Gate 0.
2. Esegui Surface Map.
3. Scegli locale/remoto.
4. Prova config nel client reale.
5. Esegui smoke test con transcript.
